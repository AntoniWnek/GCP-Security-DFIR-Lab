# Blue Team Walkthrough: Threat Hunting & Incident Response

**Objective:** To extract Indicators of Compromise (IoCs), analyze background noise (botnet activity), and reconstruct the Cyber Kill Chain using Google Cloud BigQuery and SQL log analysis.

---

## Part 1: Macro View - Threat Landscape & Background Noise
Before isolating the targeted attack, we analyze the overall external threat activity hitting the exposed DMZ to understand the background noise generated by automated scanners and botnets.

### 1.1 Top Malicious IPs Scanning the Environment
This query counts connection attempts from external IPs, filtering out internal RFC1918 traffic, to identify the most aggressive scanners.

```sql
SELECT
  jsonPayload.connection.src_ip AS attacker_public_ip,
  COUNT(*) AS total_requests,
  MIN(timestamp) AS first_attack_time,
  MAX(timestamp) AS last_attack_time
FROM
  `[YOUR_PROJECT_ID].soc_telemetry_logs.compute_googleapis_com_vpc_flows_*`
WHERE
  NOT jsonPayload.connection.src_ip LIKE '10.%'
GROUP BY
  attacker_public_ip
ORDER BY
  total_requests DESC
LIMIT 10;
```

### 1.2 Most Targeted Ports by External Threats
Identifying which services attackers are actively looking for across the external perimeter.

```sql
SELECT
  jsonPayload.connection.dest_port AS target_port,
  COUNT(*) AS connection_attempts
FROM
  `[YOUR_PROJECT_ID].soc_telemetry_logs.compute_googleapis_com_vpc_flows_*`
WHERE
  NOT jsonPayload.connection.src_ip LIKE '10.%'
GROUP BY
  target_port
ORDER BY
  connection_attempts DESC
LIMIT 5;
```

### 1.3 Data Exfiltration Volume Anomaly
Detecting potential data theft by identifying unusual outbound data transfers (bytes sent) from internal IP addresses to external destinations.

```sql
SELECT
  jsonPayload.connection.src_ip AS internal_source,
  jsonPayload.connection.dest_ip AS external_destination,
  SUM(CAST(jsonPayload.bytes_sent AS INT64)) AS total_bytes_exfiltrated,
  COUNT(*) AS session_count
FROM
  `[YOUR_PROJECT_ID].soc_telemetry_logs.compute_googleapis_com_vpc_flows_*`
WHERE
  jsonPayload.connection.src_ip LIKE '10.%'
  AND NOT jsonPayload.connection.dest_ip LIKE '10.%'
GROUP BY
  internal_source, external_destination
HAVING
  total_bytes_exfiltrated > 1000
ORDER BY
  total_bytes_exfiltrated DESC
LIMIT 5;
```

## Part 2: Micro View - Targeted Attack Reconstruction
These queries isolate the specific lateral movement campaign executed during the Red Team assessment.

### 2.1 Initial Access (Web Shell Deployment)
Searching the OS-level system logs for signs of Command Injection and the downloading of the malicious PHP payload.

```sql
SELECT
  timestamp,
  resource.labels.instance_id,
  jsonPayload.message
FROM
  `[YOUR_PROJECT_ID].soc_telemetry_logs.audit_log_*`
WHERE
  jsonPayload.message LIKE '%webshell.php%' OR jsonPayload.message LIKE '%wget%'
ORDER BY
  timestamp ASC
LIMIT 5;
```

### 2.2 Local & Network Reconnaissance
Tracking the attacker's situational awareness commands executed immediately after gaining shell access.

```sql
SELECT
  timestamp,
  resource.labels.instance_id,
  jsonPayload.message
FROM
  `[YOUR_PROJECT_ID].soc_telemetry_logs.audit_log_*`
WHERE
  jsonPayload.message LIKE '%whoami%' 
  OR jsonPayload.message LIKE '%ip route%' 
  OR jsonPayload.message LIKE '%ping -c 1%'
```

### 2.3 Credential Access & SSH Key Hijacking
Identifying the moment the attacker discovered and interacted with the unencrypted private key.

```sql
SELECT
  timestamp,
  resource.labels.instance_id,
  jsonPayload.message
FROM
  `[YOUR_PROJECT_ID].soc_telemetry_logs.audit_log_*`
WHERE
  jsonPayload.message LIKE '%id_ed25519%' 
  AND (jsonPayload.message LIKE '%cat%' OR jsonPayload.message LIKE '%ssh-keygen%')
ORDER BY
  timestamp ASC;
```

### 2.4 Lateral Movement (VPC Flow Logs)
Isolating unauthorized internal communication. This query detects traffic originating from the compromised Bastion host directly to the internal server's SSH port.

```sql
SELECT
  timestamp,
  jsonPayload.connection.src_ip AS source_ip,
  jsonPayload.connection.dest_ip AS target_ip,
  jsonPayload.connection.dest_port AS target_port,
  jsonPayload.bytes_sent
FROM
  `[YOUR_PROJECT_ID].soc_telemetry_logs.compute_googleapis_com_vpc_flows_*`
WHERE
  jsonPayload.connection.src_ip LIKE '10.%'
  AND jsonPayload.connection.dest_ip = '10.0.2.2'
  AND jsonPayload.connection.dest_port = 22
ORDER BY
  timestamp ASC;
```
